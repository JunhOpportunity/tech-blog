## 동시성 & 멀티쓰레딩

> 자바스크립트는 싱글쓰레드이지만 비동기 방식을 통해서 동시성을 달성할 수 있다.
> 
- 멀티스레드 : 병렬적으로 실행되는 것 처럼 보이지만, 싱글코어에서는 운영체제가 빠르게 번갈아 실행한다. (멀티코어에서는 병렬로 가능하긴 함)
    - 단점 : 여러 스레드가 동시에 공유 자원에 접근하려 할 때 데드락 현상 발생. 디버깅과 테스트가 어려움. 메모리 사용 증가
- 동기 : 코드가 순서대로 실행
- 비동기 : 작업이 완료될 때까지 기다리지 않고 다음 코드 실행
