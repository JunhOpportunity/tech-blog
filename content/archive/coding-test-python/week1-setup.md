## TIP

- _.isalpha() : 알파벳인지 아닌지 검사하는 것
- ASCII 코드 : A - 65, a - 97
- ord(’a’) : 알파벳을 아스키 코드로 변환
- chr(97) : 아스키 코드를 알파벳으로 변환
- string[1:-1] : string의 0번째 인덱스와 마지막 인덱스를 제외하고 반환한다 ex) ABCDE ⇒ BCD
